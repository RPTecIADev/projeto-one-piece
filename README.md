## 🌊 Sobre o Projeto

Este projeto é uma **landing page imersiva** baseada no universo de One Piece. O objetivo foi criar uma experiência de usuário (UX) fluida, utilizando técnicas avançadas de manipulação de DOM para alternar entre personagens, alterando dinamicamente o fundo, as informações e a estética visual da página.

[Página](https://rpteciadev.github.io/projeto-one-piece/)

### 🚀 Funcionalidades Principais
*   **Troca de Personagens:** Seleção interativa que altera o estado visual da aplicação.
*   **Design Responsivo:** Adaptado para diferentes resoluções (Den Den Mushi friendly!).
*   **Estética Cyber-Pirata:** Mistura de elementos clássicos do anime com acabamentos em neon e efeitos de transição suaves.

## 📸 Demonstração Visual

> [!TIP]
> ### 🖱️ Interatividade em Foco
> <details>
> <summary><b>Clique para ver como funciona a troca de Haki (Personagens)</b></summary>
> 
> O projeto utiliza um sistema de classes em JavaScript que remove a classe `selecionado` do item anterior e a aplica ao novo alvo. Isso dispara transições em CSS para:
> 1. Alterar a imagem de fundo com efeito de *fade*.
> 2. Atualizar o título e a descrição do personagem.
> 3. Adicionar brilho neon ao ícone selecionado.
> </details>
